Semaine 1-2 — Actions urgentes (impact fort, effort faible)



    CRITIQUE       Harmoniser les informations NAP

  Utiliser le même téléphone et email sur TOUTES les pages (mentions légales, contact, schema).




    CRITIQUE       Ajouter meta description sur toutes les pages

  Rédiger une meta description unique (150-160 car.) par page avec mots-clés ciblés.




    CRITIQUE       Optimiser les titles

  Intégrer mots-clés (liqueur, artisanale, plantes, Haute-Loire) dans chaque title.




    CRITIQUE       Ajouter les balises canonical

  Balise <link rel="canonical"> sur chaque page pour éviter le duplicate content.




    CRITIQUE       Ajouter Open Graph + Twitter Cards

  og:title, og:description, og:image, og:url + twitter:card sur toutes les pages.




    CRITIQUE       Ajouter H1 manquants

  /nous-contacter/ et /lumiere-obscure/ n'ont pas de H1.




    CRITIQUE       Exclure pages utilitaires du sitemap

  Retirer /cart/, /checkout/, /my-account/ + ajouter noindex.
